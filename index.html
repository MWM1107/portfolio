Custom IoT Pipeline

  • Hardware: An ESP32 microcontroller wired to BME680 (Climate) and CCS811 (Gas) sensors, running custom C++ firmware.
  • Backend: The ESP32 syncs with an NTP time server and pushes UNIX-timestamped JSON payloads directly to a Firebase Realtime Database via REST API.
  • Frontend: This web client uses vanilla JavaScript to asynchronously fetch and parse the live Firebase data every 30 seconds.
