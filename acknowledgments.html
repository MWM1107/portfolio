Generative AI Disclosure

Per the course disclosure requirements, here's an honest breakdown of how I used AI tools when building this site. The short version: the design, content, and architecture are mine. AI handled most of the CSS and the JavaScript widget.

My Contributions

  • Product Vision & Design: Conceptualized the Apple-inspired aesthetic and the bento-box grid layout. Hand-drew the initial wireframes.
  • Architecture: Engineered the core HTML structure and structural relationships between pages.
  • Integration Logic: Formulated the architectural decision to route the contact page through Formspree's API rather than using a standard (and clunky) mailto: tag.
  • Content & Assets: Wrote all personal copy, project descriptions, and provided all original application icon assets.

AI Contributions

  • CSS Styling: Generated the majority of the CSS rules, specifically the complex backdrop-filters required for the glassmorphism effects and the responsive flexbox/grid alignments.
  • JavaScript Logic: Authored the asynchronous fetch requests and DOM manipulation logic for the real-time Air Quality Monitor widget.
  • Refinement: Assisted in debugging CSS mobile responsiveness issues.
